# BARASA JUMA JAMIL | FULL-STACK WEB DEVELOPER | Portfolio

This is the professional portfolio website for BARASA JUMA JAMIL, a Software Developer / UI/UX specialist. Built with React.js and TailwindCSS, fully responsive, visually appealing, and designed to impress top-tier employers.

## Features
- Full-screen hero section
- About, Skills, Experience, Projects, Education, Testimonials, Contact, and Blog sections
- Smooth scroll navigation with active highlighting
- Hover animations and section entrance animations
- Dark/light mode toggle
- Mobile responsive
- Contact form with email validation and EmailJS integration
- Modern fonts and color scheme

## Getting Started
1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm start
   ```

## Customization
- Replace [Your Name], [Your Industry/Role], and content in each section with your own details.
- Update project images and links in `Projects.js`.
- Configure EmailJS for the contact form.

## Deployment
Build for production:
```
npm run build
```
